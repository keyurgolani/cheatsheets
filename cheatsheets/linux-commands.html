üìÅ File & Directory Operations
ls

find . -name "*.txt"
Find files by name pattern

üîê Permissions & Ownership
chmod 755 [file]
Set permissions (rwxr-xr-x)
chmod +x [file]
Make file executable
chmod -R 644 [dir]
Set permissions recursively
chown user [file]
Change file owner
chown user:group [file]
Change owner and group
chgrp [group] [file]
Change file group
umask 022
Set default permissions for new files
Permission Numbers:
4 = read (r), 2 = write (w), 1 = execute (x)
755 = rwxr-xr-x | 644 = rw-r--r-- | 777 = rwxrwxrwx

üîó Pipes & Redirection

Pipe output of cmd1 to cmd2

Redirect output to file (overwrite)

cmd < file

‚è∞ Scheduling & Cron

*/15 * * * * /check.sh = Run every 15 minutes
üí° Common Command Combinations
$ find . -type f -name "*.log" -mtime +30 -delete

Delete log files older than 30 days

$ ps aux | grep python | grep -v grep | awk '{print $2}' | xargs kill

Kill all Python processes

$ du -sh */ | sort -h

Show directory sizes sorted

$ tail -f /var/log/syslog | grep --line-buffered "error"

Monitor log file for errors in real-time

$ find . -type f -exec grep -l "TODO" {} \;

Find all files containing "TODO"
