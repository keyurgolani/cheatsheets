Stream editing and pattern scanning essentials

üîß sed Basics
sed 's/old/new/' file
Replace first occurrence per line
sed 's/old/new/g' file
Replace all occurrences (global)
sed -i 's/old/new/g' file
Edit file in-place
sed -n 'p' file
Print (suppress automatic output with -n)
sed 'd' file
Delete all lines
üìç sed Line Addressing
sed '5d' file
Delete line 5
sed '2,5d' file
Delete lines 2 through 5
sed '1,/pattern/d' file
Delete from line 1 to first pattern match
sed '/pattern/d' file
Delete lines matching pattern
sed -n '10,20p' file
Print only lines 10-20
sed -n '/start/,/end/p' file
Print from start pattern to end pattern
‚úèÔ∏è sed Substitution
sed 's/old/new/2' file
Replace 2nd occurrence on each line
sed 's/old/new/gi' file
Global, case-insensitive replace
sed 's|/path/old|/path/new|' file
Use | delimiter for paths
sed 's/^/prefix/' file
Add prefix to each line
sed 's/$/suffix/' file
Add suffix to each line
sed 's/.*pattern.*/replacement/' file
Replace entire line containing pattern
üîÑ sed Advanced Commands
sed 'y/abc/ABC/' file
Transliterate (like tr command)
sed '/pattern/a\new line' file
Append line after pattern
sed '/pattern/i\new line' file
Insert line before pattern
sed '/pattern/c\replacement' file
Change (replace) matching line
sed -e 'cmd1' -e 'cmd2' file
Multiple commands
sed 's/\([0-9]*\)/[\1]/' file
Capture groups with \( \) and reference with \1
üìä awk Basics
awk '{print}' file
Print all lines
awk '{print $1}' file
Print first field
awk '{print $1, $3}' file
Print fields 1 and 3
awk '{print $NF}' file
Print last field
awk -F: '{print $1}' file
Use : as field separator
üîç awk Pattern Matching
awk '/pattern/' file
Print lines matching pattern
awk '!/pattern/' file
Print lines NOT matching pattern
awk '/start/,/end/' file
Print from start to end pattern
awk '$1 == "value"' file
Print if field 1 equals value
awk '$3 > 100' file
Print if field 3 is greater than 100
awk '$2 ~ /pattern/' file
Print if field 2 matches pattern
üßÆ awk Built-in Variables
NR
Current record (line) number
NF
Number of fields in current record
FS
Field separator (default: whitespace)
OFS
Output field separator
RS
Record separator (default: newline)
ORS
Output record separator
FILENAME
Current filename
üî¢ awk Calculations
awk '{sum += $1} END {print sum}' file
Sum of field 1
awk '{sum += $1} END {print sum/NR}' file
Average of field 1
awk '{print $1 * $2}' file
Multiply fields 1 and 2
awk '{total += $3} END {print total}' file
Accumulate field 3
üéØ awk BEGIN & END
awk 'BEGIN {print "Header"} {print} END {print "Footer"}' file
Execute before, during, and after processing
awk 'BEGIN {FS=":"} {print $1}' file
Set field separator in BEGIN
awk 'BEGIN {OFS="|"} {print $1, $2}' file
Set output field separator
üìù awk String Functions
awk '{print length($1)}' file
Length of field 1
awk '{print toupper($1)}' file
Convert to uppercase
awk '{print tolower($1)}' file
Convert to lowercase
awk '{print substr($1, 1, 3)}' file
Substring: chars 1-3
awk '{gsub(/old/, "new"); print}' file
Global substitution
awk '{sub(/old/, "new"); print}' file
First occurrence substitution
üîÄ awk Control Flow
awk '{if ($1 > 10) print $1}' file
If statement
awk '{if ($1 > 10) print "big"; else print "small"}' file
If-else statement
awk '{for(i=1; i<=NF; i++) print $i}' file
For loop through fields
awk '{while (x<5) {print x; x++}}' file
While loop
üìä awk Arrays
awk '{count[$1]++} END {for (w in count) print w, count[w]}' file
Count occurrences of field 1
awk '{arr[NR] = $1} END {print arr[1]}' file
Store in array, access later
