üîç Example: JMESPath Queries
# Get all running instance IDs

--output text

--output table

--query "Buckets[?CreationDate>=`2024-01-01`].Name"
üí° Useful Tips
Profiles: Manage multiple AWS accounts
‚Ä¢ Configure: aws configure --profile prod
‚Ä¢ Use: aws s3 ls --profile prod
‚Ä¢ Set default: export AWS_PROFILE=prod
Output Formats: Choose the right format
‚Ä¢ JSON (default): --output json
‚Ä¢ Table: --output table (human-readable)
‚Ä¢ Text: --output text (scripting)
‚Ä¢ YAML: --output yaml
Pagination: Handle large result sets
‚Ä¢ Auto-pagination is enabled by default
‚Ä¢ Disable: --no-paginate
‚Ä¢ Manual: --max-items 100 --starting-token [token]
Autocomplete: Enable CLI autocomplete
‚Ä¢ Bash: complete -C aws_completer aws
‚Ä¢ Zsh: Add to ~/.zshrc
‚ö†Ô∏è Warning: Always use --dry-run to test destructive commands before execution. Check your region and profile settings!
