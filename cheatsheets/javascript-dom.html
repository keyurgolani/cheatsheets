üîç Element Selection
document.getElementById('id')
Select element by ID
document.querySelector('.class')
Select first matching element (CSS selector)
document.querySelectorAll('div')
Select all matching elements (returns NodeList)

Select elements by class name (returns HTMLCollection)

Select elements by tag name

üí° querySelector and querySelectorAll are more flexible and use CSS selector syntax!

üí° Common Patterns
// Pattern 1: Create and append multiple elements
const fragment = document.createDocumentFragment();
for (let i = 0; i < 100; i++) {
const li = document.createElement('li');
li.textContent = `Item ${i}`;
fragment.appendChild(li);
}
list.appendChild(fragment); // Single reflow!
// Pattern 2: Event delegation for dynamic content
document.querySelector('.container').addEventListener('click', (e) => {
if (e.target.classList.contains('delete-btn')) {
e.target.closest('.item').remove();
}
});
// Pattern 3: Toggle visibility
const toggle = (element) => {
element.style.display = element.style.display === 'none' ? 'block' : 'none';
};
// Pattern 4: Wait for DOM to load
document.addEventListener('DOMContentLoaded', () => {
// DOM is ready, safe to manipulate
console.log('DOM loaded');
});
// Pattern 5: Debounce scroll/resize events
let timeout;
window.addEventListener('resize', () => {
clearTimeout(timeout);
timeout = setTimeout(() => {
console.log('Resize ended');
}, 250);
});
// Pattern 6: Check if element is in viewport
const isInViewport = (element) => {
const rect = element.getBoundingClientRect();
return (
rect.top >= 0 &&
rect.left >= 0 &&
rect.bottom <= window.innerHeight &&
rect.right <= window.innerWidth
);
};
‚ö° Performance Tips
üí° Minimize Reflows: Batch DOM changes using DocumentFragment or by modifying elements before adding them to the DOM.
üí° Event Delegation: Attach event listeners to parent elements instead of individual children for better performance with dynamic content.
üí° Cache Selectors: Store frequently accessed elements in variables instead of querying the DOM repeatedly.
üí° Use classList: Prefer classList methods over direct className manipulation for better readability and performance.
‚ö†Ô∏è Avoid innerHTML for User Input: Use textContent or createElement to prevent XSS attacks.
