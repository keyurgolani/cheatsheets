# Contributing to Cheatsheets Collection

Thank you for your interest in contributing! This guide will help you add new cheatsheets to the collection.

## How to Contribute

### Adding a New Cheatsheet

1. **Fork and clone** this repository
2. **Create your cheatsheet** as an HTML file in the `cheatsheets/` directory
   - Use a descriptive filename (e.g., `git-commands.html`, `python-basics.html`)
   - Follow the existing HTML structure for consistency
3. **Add a link** to your cheatsheet in `index.html`
   - Add it to the cheatsheets section
4. **Test locally** by opening `index.html` in your browser
5. **Submit a pull request** with a clear description of your cheatsheet

### Cheatsheet Guidelines

- Keep content concise and focused
- Use clear examples
- Ensure proper HTML formatting
- Test that all links work correctly

## Questions?

Feel free to open an issue if you have questions or suggestions.
